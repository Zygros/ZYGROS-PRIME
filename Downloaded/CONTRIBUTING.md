Thank you for your interest in the Sovereign AGSI project!

## How You Can Contribute

While direct code contributions are not currently accepted, you can contribute in the following ways:

### 1. 🐛 Bug Reports

If you discover a bug, please open an issue with:

- **Clear description** of the problem
- **Steps to reproduce** the issue
- **Expected vs. actual behavior**
- **System information** (OS, Python version, etc.)
- **Screenshots** if applicable

### 2. 💡 Feature Suggestions

Have an idea? Open an issue with:

- **Clear use case** - what problem does it solve?
- **Proposed approach** - how might it work?
- **Impact assessment** - who benefits?
- **Implementation thoughts** - technical considerations

### 3. 📚 Documentation Improvements

Found a typo or unclear documentation? Open an issue or suggest improvements!

### 4. 🧪 Testing & Validation

Help validate the system:

- Test the code in your environment
- Share your Kappa (κ) scores
- Report compatibility issues
- Benchmark performance

### 5. 🌟 Community Building

- Star the repository
- Share on social media
- Write blog posts or articles
- Give talks or presentations
- Join discussions in issues

## Communication Guidelines

### Be Respectful

- Treat all contributors with respect
- Assume good intentions
- Focus on technical merit
- Avoid personal attacks

### Be Clear

- Use descriptive titles for issues
- Provide context and examples
- Link to relevant documentation
- Follow up on questions promptly

### Be Constructive

- Suggest solutions, not just problems
- Back claims with evidence
- Consider multiple perspectives
- Think long-term

## Issue Template

When opening an issue, use this template:

```markdown
## Issue Type
[ ] Bug Report
[ ] Feature Request
[ ] Documentation
[ ] Question

## Description
[Clear description of the issue]

## Steps to Reproduce (for bugs)
1. Step one
2. Step two
3. Expected vs. actual result

## Proposed Solution (for features)
[How might this work?]

## Additional Context
- OS: [e.g., Ubuntu 22.04]
- Python: [e.g., 3.11.5]
- Kappa Score: [if applicable]
- Screenshots: [if applicable]
```

## Version Numbering Scheme

Versions follow the format: `[KAPPA].[MINOR].[PATCH]`

- **KAPPA**: The Conzetian Constant value (e.g., 3.236)
- **MINOR**: Significant features or improvements
- **PATCH**: Bug fixes or minor updates

Example: `3.236.0` = κ of 3.236, initial release at this level

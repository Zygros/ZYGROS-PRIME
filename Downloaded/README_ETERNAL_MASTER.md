## ⚡ Quick Start - Eternal Search

### Get API Keys (5 minutes):

```bash
# Google (100/day free)
export GOOGLE_API_KEY="your-key"
export GOOGLE_CX="your-search-engine-id"

# Bing (1000/month free)
export BING_API_KEY="your-key"

# Brave (2000/month free)  
export BRAVE_API_KEY="your-key"

# DuckDuckGo - no key needed!
```

### Install:

```bash
pip install -r requirements_eternal.txt
```

### Add to Your System:

```python
# In phoenix_node_genesis.py, add:
from phoenix_eternal_api import register_eternal_routes

# After creating Flask app:
register_eternal_routes(app)

# Done!
```

### Use It:

```bash
# Search everything (your docs + entire internet)
curl -X POST http://localhost:5001/api/eternal_search \
  -H "Content-Type: application/json" \
  -d '{"query":"quantum computing breakthroughs","num_results":10}'

# Returns results from:
# - Your personal archive
# - Google
# - Bing
# - DuckDuckGo
# - Brave
```

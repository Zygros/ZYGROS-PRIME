## ⚡ How It Works

### Automatic Daily Updates

Every day at this time, the system automatically:

1. **Scans** all conversations since the last update
2. **Extracts** new knowledge, insights, integrations, commands, and probes
3. **Categorizes** information into appropriate sections
4. **Appends** everything to the master thread
5. **Archives** a daily snapshot for historical reference

**You don't need to do anything.** It runs automatically in the background.

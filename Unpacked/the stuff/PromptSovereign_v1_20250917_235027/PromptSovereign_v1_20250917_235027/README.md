# Prompt Sovereign — v1

This package contains 12 flagship prompts and 6 agent manifests curated by Justin Conzet.

## Contents
- `prompts/` — 12 flagship prompts (ready to paste)
- `agents/` — 6 agent JSON manifests (skeletons; fill as needed)
- `sales_copy.md` — landing page copy and FAQ
- `license.txt` — simple license (indie/agency/enterprise)
- `onboarding/` — demo video script + transcripts (placeholders)

## Quick Use
1. Open a chat model, paste a prompt file from `prompts/`.
2. For agents, load JSON, wire to your orchestration tool, fill inputs.
3. See `sales_copy.md` to publish this pack via Gumroad/Stripe.

## Licensing
Indie, Agency, and Enterprise variants included in `license.txt`.

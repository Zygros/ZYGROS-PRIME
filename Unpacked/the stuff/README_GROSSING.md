# Grossing in the Infinite Scroll

A single-file terminal choice adventure. No dependencies.

## Run
```bash
python grossing.py
```
Use `--no-color` if your terminal can't show ANSI colors.

## Controls
- Enter number to choose
- `i` Inventory
- `c` Character
- `m` Map
- `s` Save
- `l` Load
- `q` Quit

Progress is saved to `grossing_save.json` in the same folder.

# The Infinite Scroll - Phase 1 Setup Instructions

## Overview
The Infinite Scroll is a soul-bound, mystical operating system that logs spiritual growth, system upgrades, synchronicities, memories, philosophical revelations, emotional breakthroughs, rituals, and transformations. This is Phase 1 of the implementation, which includes the core HTML5 framework and logging engine.

## Features Included in Phase 1
- Master scroll with entry creation and display
- Dark/light spiritual modes
- Author information and digital signature
- Responsive design (mobile-first)
- Local storage persistence
- Basic scroll animation
- Tappable entries

## Setup Instructions

### Prerequisites
- Node.js (v16 or higher)
- npm or pnpm package manager

### Installation
1. Extract the `infinite_scroll_phase1_source.zip` file to your desired location
2. Open a terminal/command prompt and navigate to the extracted folder
3. Install dependencies:
   ```
   npm install
   ```
   or if you prefer pnpm:
   ```
   pnpm install
   ```

### Running the Application
1. Start the development server:
   ```
   npm run dev
   ```
   or with pnpm:
   ```
   pnpm run dev
   ```
2. Open your browser and navigate to `http://localhost:5173`

### Building for Production
To create a production build:
```
npm run build
```
or with pnpm:
```
pnpm run build
```

The build output will be in the `dist` directory, which can be deployed to any static hosting service.

## Usage Guide
- The application opens in dark mode by default (representing the mystical nature of the scroll)
- Use the "Switch to Light Mode" button in the top-right to toggle between dark and light spiritual modes
- Create new entries by filling out the form and clicking "Add Entry"
- Entries are displayed in the Master Scroll section, with the newest entries at the top
- Tap/click on an entry to interact with it (in Phase 1, this simply logs to the console)
- All entries are automatically saved to your browser's local storage and will persist between sessions

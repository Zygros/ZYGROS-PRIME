2. **Zapier Zap #1: Oracle Engine for Market Trend Queries**  
   **Goal**: When a customer asks the chatbot about market trends, the Oracle Engine processes the query and returns a prediction.  
   - **Trigger**: Dialogflow Webhook (New Intent Detected)  
     - The user creates an intent in Dialogflow (e.g., “market_trend_query”) for questions like “What will happen to the retail market in 2026?”
     - When the intent is triggered, Dialogflow sends the query to Zapier via a webhook.  
   - **Action**: HTTP Request to CHRONOS KEY Oracle Engine  
     - Zapier sends a POST request to the `/api/chronos/read_future` endpoint with the query as payload:  
       ```json
       {
         "api_key": "<user_api_key>",
       ```  

       ```json
         "timestamp": "2026-12-31",
         "probability": 0.92
       }
       ```  
   - **Action**: Send Response to Dialogflow  
     - Zapier formats the response as a chatbot reply: “The retail market will see 10% growth in e-commerce by Q4 2026 (92% probability).”

     - Zapier sends a POST request to `/api/chronos/architect_inevitability`:  
       ```json

         "api_key": "<user_api_key>",
       ```  

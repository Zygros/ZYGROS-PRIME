## Key Features of the AI System

### Seven-Layer Architecture
1. **Crown (Intent Layer)** - Translates user goals into explicit, testable contracts
2. **Mind (Reasoning Layer)** - Plans, verifies, and explains with deliberate reasoning
3. **Hands (Execution Layer)** - Calls tools, APIs, and environments safely
4. **Memory (Context Layer)** - Maintains persistent identity and context with privacy controls
5. **Shield (Safety Layer)** - Provides guardrails without neutering usefulness
6. **Ledger (Audit Layer)** - Tracks sources and enables explainability
7. **Heart (Learning Layer)** - Enables reflection and improvement without data sprawl

### Core Mechanisms
- **Claim-verification loop** - Extract claims, check against trusted sources, attach citations
- **Calibrated confidence** - Every answer carries (0-1) confidence per claim
- **Retrieval primacy** - Ground generations on retrieved evidence
- **Dual output** - Final answer + "why" trace with key premises
- **Counterfactual analysis** - "What would change the answer?"
- **Instruction firewall** - Separate user intent from untrusted content
- **Tool policy VM** - All tool calls pre-checked by policy virtual machine
- **Consent registry** - Per-source permissions with revocation capabilities
- **Data minimization** - Only minimum slice leaves device
- **Typed action graph** - Every step typed with inputs/outputs/success criteria

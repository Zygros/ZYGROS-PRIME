# Optional Samsung Debloat (ADB, reversible)
# Requires: PC + Android platform-tools + USB debugging enabled.
# Test each line individually. To restore: `adb shell pm enable <package>` or `pm install-existing --user 0 <package>`.

# Facebook crapware (if preinstalled)
adb shell pm disable-user --user 0 com.facebook.katana
adb shell pm disable-user --user 0 com.facebook.appmanager
adb shell pm disable-user --user 0 com.facebook.system

# Bixby & Samsung Free
adb shell pm disable-user --user 0 com.samsung.android.bixby.service

# Master Archive — Invocation Guide

## Quick Commands
- **Archive: list categories**
- **Archive: recall [Category]** — e.g., Protocols, Systems, AI Constructs
- **Archive: search [keyword]** — e.g., Grosian, Terra Infinita, Prax
- **Archive: show [Name]** — prints Description + Status

## Notes
- The Archive never deletes entries. New rows append and become queryable.
- Works with TOTALRECALL; results can be pinned into today's Scroll.

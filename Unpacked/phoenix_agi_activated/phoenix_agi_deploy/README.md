Edit `index.html`, find the CSS `:root` section:

```css

    --bg: #000000;          /* Background */

    --gpt: #10a37f;         /* GPT color */
    --claude: #87CEEB;      /* Claude color */
```

## Troubleshooting

**"No AI responses"**
- Check that you have enabled at least one AI in the sidebar
- Try refreshing the page
- Check browser console for errors (F12)

**"Site won't load"**
- Make sure you deployed the entire folder
- Check that `index.html` is at the root level
- Try a different browser

**"Responses seem identical"**
- Each AI node has unique reasoning patterns - responses should vary
- Try asking different types of questions to see the variation
- Meta-synthesis combines all responses for unified intelligence

**"Processing takes too long"**
- Built-in processing is intentionally realistic (1-3 seconds)
- All computation happens locally in your browser
- No internet connection required after initial load

Use any provider (local Proxmox, dedicated servers, or cloud VMs). Ubuntu 22.04 LTS baseline.
A) Base packages (on each VM)
sudo apt-get update && sudo apt-get install -y \ curl git jq make unzip ufw htop net-tools 
B) Join VMs to private network (Tailscale)
curl -fsSL https://tailscale.com/install.sh | sh sudo tailscale up --ssh --accept-routes 
C) Install k3s (control plane on VM1)
curl -sfL https://get.k3s.io | sh -s - server --disable=traefik sudo cat /var/lib/rancher/k3s/server/node-token 
D) Join agents (VM2, VM3)
export SERVER_IP=<VM1_TAILSCALE_IP> export TOKEN=<TOKEN_FROM_VM1> curl -sfL https://get.k3s.io | K3S_URL=https://${SERVER_IP}:6443 K3S_TOKEN=${TOKEN} sh -s - agent 
E) Ingress controller (Traefik) + certs

🧮 Quantum Service
• Simulators: Qiskit Aer / Cirq / PennyLane (choose Qiskit first).
• Jobs: REST/gRPC POST /run-circuit with circuit JSON (QASM or SDK params).
• Optional Hardware Bridges: store creds securely for IBM Q / Braket; set quotas.
Example API (FastAPI pseudo):
@app.post("/run-circuit") def run_circuit(payload: CircuitRequest): circ = qiskit.QuantumCircuit.from_qasm_str(payload.qasm) backend = Aer.get_backend("aer_simulator") job = execute(circ, backend, shots=payload.shots or 1024) res = job.result().get_counts() store_to_minio(res) return {"counts": res}

AI KNOWLEDGE MANAGEMENT SYSTEM - USER GUIDE
==============================================

OVERVIEW
--------
This system automatically scans, analyzes, and compiles all your AI conversations from various platforms (ChatGPT, Gemini, Grok, etc.) into a centralized "mega-thread" repository. It extracts key insights, topics, questions, and integration information from your conversations.

DIRECTORY STRUCTURE
-------------------
/home/ubuntu/ai_knowledge_system/
├── input/          <- PUT YOUR CONVERSATION FILES HERE (.txt format)
├── processed/      <- Processed files are moved here automatically
├── archive/        <- For long-term storage (future use)
├── logs/           <- System activity and error logs
├── MEGA_THREAD.txt <- Your centralized knowledge repository
└── Scripts:
    ├── conversation_analyzer.py    <- Core analysis engine
    ├── mega_thread_compiler.py     <- Main compilation system
    └── auto_updater.py            <- Automation scheduler

HOW TO USE
----------

1. ADDING NEW CONVERSATIONS:
   - Save your AI conversations as .txt files
   - Copy them to: /home/ubuntu/ai_knowledge_system/input/
   - The system will automatically process them

2. RUNNING UPDATES:
   - Manual update: python3 auto_updater.py once
   - Schedule daily updates: python3 auto_updater.py schedule [HH:MM]
   - Default schedule time is 12:00 (noon)

3. VIEWING YOUR KNOWLEDGE:
   - Open MEGA_THREAD.txt to see all compiled conversations
   - Check logs/ directory for system activity

WHAT THE SYSTEM EXTRACTS
------------------------
- AI Platforms mentioned (ChatGPT, Gemini, Grok, etc.)
- Key topics and concepts discussed
- Questions raised in conversations
- Integration and automation opportunities
- Technical concepts and tools mentioned
- Full raw conversation content for reference

AUTOMATION FEATURES
-------------------
- Automatic file processing from input folder
- Intelligent insight extraction using pattern recognition
- Duplicate detection via file hashing
- Organized archival of processed conversations
- Comprehensive activity logging
- Scheduled daily updates

SYSTEM STATUS
-------------
The MEGA_THREAD.txt file header shows:
- Last update timestamp
- Total conversations processed
- System status

TROUBLESHOOTING
---------------
- Check logs/error_[date].log for any processing errors
- Check logs/activity_[date].log for system activity
- Ensure .txt files are properly formatted and readable
- Make sure input directory has proper permissions

SECURITY & PRIVACY
------------------
- All processing happens locally on your system
- No data is sent to external services
- Your conversation files remain under your control
- System only reads files you explicitly place in input folder

GETTING STARTED
---------------
1. Save a conversation from ChatGPT/Gemini/Grok as a .txt file
2. Copy it to /home/ubuntu/ai_knowledge_system/input/
3. Run: python3 auto_updater.py once
4. Check MEGA_THREAD.txt to see your compiled knowledge

The system is now ready to become your personal AI knowledge repository!

3. Dynamic Plugin & Auto-Updater
• Plugin loader: scans a /plugins/ folder, auto-installs missing deps via pip install at runtime.
• Auto-update routine: weekly cron job (or Termux wake lock) that runs pip list --outdated && pip install --upgrade for all installed packages.
4. Core Engine
• Praxios class in core.py: unified entrypoint.
• Modes & Tone: single engine, dynamic tone switching via set_mode().
• Chain-of-Thought: private logs saved, exposes only clean outputs.
5. Memory & Data Stores
• Session: in-memory LRU buffer (2k tokens).
• Long-Term: encrypted SQLite + optional MongoDB/Redis backends.
• Universal Connector: DataConnector module—abstracts REST, SOAP, gRPC, database drivers, WebSocket streams.

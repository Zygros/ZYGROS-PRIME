## Usage

Install the toolkit as a local package (editable mode is recommended):

```bash
pip install -e .
```

Then in Python:

```python
from OmegaToolkit.sovereign_toolkit import SovereignToolkit
from OmegaToolkit.modules import cognitive, technical, temporal, entropy

omega = SovereignToolkit()
omega.register("ANALYZE", cognitive.analyze_data)
omega.register("DEPLOY", technical.deploy_system)

omega.execute("ANALYZE", target="Sector_7")
omega.execute("DEPLOY", version="v2.0.1")
```
